# Not accepting contributions

This repository is a snapshot demo of the erbswap pattern, not an actively
maintained library.

**Forks are welcome.** The code is MIT-licensed — clone, copy, vendor, or fork
freely. If you want to extend the pattern, the right move is to do it in your
own fork.

**Pull requests against this repo are not reviewed.** I'm not maintaining a PR
queue here. The same applies to issues — they may go unanswered.

For the pattern's philosophy and the four invariants that bound its scope, see
[docs/INVARIANTS.md](docs/INVARIANTS.md).
